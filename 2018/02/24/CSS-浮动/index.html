CSS 浮动

  • 会使元素向左或向右移动，其周围的元素也会重新排列。浮动的框可以向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止。由于浮动框不在文档的普通流中，所以文档的普通流中的块框表现得就像浮动框不存在一样。

CSS 浮动属性

属性 描述 值
clear 指定不允许元素周围有浮动元素。 left,right,none,inherit
float 指定一个盒子（元素）是否可以浮动。 left,right,none,inherit
