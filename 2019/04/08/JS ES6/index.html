函数的方法bind()

  • bind()方法会创建一个新的函数，称为绑定函数，当调用这个函数的时候，绑定函数会以创建它时传入bind()方法的第一个参数作为this,传入bind()方法的第二个以及以后的参数加上绑定函数运行时本身的参数按照顺序作为原函数来调用原函数
  • function fn(){}

  • fn.bind()

  • 函数.bind(参数)

  • 参数用来改变原函数的指向，返回新的函数，新函数中的this指向这个函数
  • 函数.bind(参数1，参数2…….)

  • 参数1用来改变原函数的this的指向，返回新的函数，新的函数中的this指向这个参数1，往后所有的参数，用来改变之后的新函数中传参

  • 执行结果：

  • 函数：bind(参数);执行之后，得到的返回值，是新函数
  • 这个新函数，是被改变后的老函数

ES6新增声明变量的方式

  • let关键字用来声明变量，类似于var,但是使用let变量有以下几点特征：
  1. 不存在变量提升
  2. 暂时性死区
  3. 不允许重复声明，同一作用域只能声明一次
  4. 新增块级作用域
  • for(作用域A){作用域B}循环的应用；
  • const关键字用来声明变量，其使用规则除了声明之后不能修改之外，其他特性和let一样
  • 本质上来说，const声明的变量并不是变量的值不能随便改动，而是变量指向的内存地址不得改动，简单的数据类型(数字，字符串，布尔)，值就保存在变量指向的那个内存地址，因此等同于常量，但是对于复合型的数据(对象，数组),变量指向的内存地址保存的只是一个指针，const只能保证这个指针是固定的，至于指向的内部结构就不能控制到了。

JSON.parse和JSON.stringify

  • JSON是一种高效的数据存储格式，JSON的结构和对象一致，也是以键值对的形式来进行存储的;
  • ES5中新增对象JSON的有序化和反序化方法
  • JSON.parse(str) 将字符串转换成JSON
  • JSON.stringify(obj) 将JSON转换成字符串
  • 对象的key必须放在双括号内var obj={“left”:10}
  • json的字符串必须是单引号var str=’{“left”:10}’
  • 在JSON文件中，不允许出现undefined和function和NaN,在强制类型转换的过程中会省略undefined和function的属性，将NaN转换成null
