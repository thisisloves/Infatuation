正则表达式

  • 正则的创建方式
  1. 构造函数的方式

2.字面量的方式，古老的prel语言风格

  • //是正则表达式的标识符
  • “”是字符串的标识符
  • []是数组的标识符

  • {}是对象的标识符

  • 正则中的修饰符，写在正则表达式后面/的后面

  • g表示全局匹配，查找所有

  • i表示忽略大小写

  • 正则的验证

  • var reg=/[0-9]/;
  • reg.test(str); //返回为true表示验证成功，false表示失败

字符串.repalace(oldstr,newstr)函数和字符串.match(正则)函数

  • 字符串的replace的函数的功能:将老字符串转换成正则表达式，对字符串进行替换

  • 字符串的match函数的功能提取所有想要的东西，返回一个数组，包含所有符合条件的字符

  • var str=”abc123abc456AdjisjDBDHHcsdccsaa”;

  • \d 匹配所有的数字
  • match函数的功能:提取所有想要的东西,返回一个数组,包含所有符合条件的字符
  • var reg=/\d/g;
  • console.log(str.match(reg)); //匹配所有数字并以数组输出，不连续
  • var reg=/\d+/g;
  • console.log(str.match(reg)) //打印字符串中所有的数字,连续
  • |或，类似于js中的||
  • var reg=/abc|bbc|cbc|dbc/g;
  • console.log(str.match(reg)); //打印字符串中的abc,bbc,cbc,dbc
  • 中元符[]
  • var reg=/[a-z]/ //匹配所有字母
  • ^ 排除(除了)
  • var reg=/[^0-9]/ //匹配表示除了字母之外的所有
  • . 点 代表所有,不建议使用
  • var reg=/<.+>/g;
  • console.log(str.replace(reg,””)); //过滤尖括号里面的东西
  • 正则表达式的贪婪定律
  • var reg=/<[^<>]+>/g

  • console.log(str.replace(reg,””)); //表示尖括号里面的东西都不要

  • 转义字符:

  • \d [0-9] 数字
  • \w [a-z0-9_-] 数字字母,下划线

  • \s 空白字符

  • \D [^0-9] 非数字

  • \W [^a-z0-9_] 非数字,字母,下划线

  • \S 非空白字符

  • 量词:限定符,可以指定正则表达式的一个给定组,必须要出现多少次才能满足匹配

  • ? 匹配前面的子表达式是零次或者一次
  • {n} 匹配n次
  • {n,}至少匹配n次

  • {n,m}至少匹配n次,最多匹配m次

  • 注意:在限定符后面跟?,则由贪婪匹配变成非贪婪匹配

  • 正则的方法

  • 正则.test(字符串)方法,返回值是true和false
  • 正则.test(字符串)有一个特性,就是只要字符串的一部分符合要求那么就会返回true
  • 解决方法:
  • ^ 开始
  • $ 结束
