答题技巧： 方法有没有参数，传几个参数，传不同的参数，有什么作用。（思考跟有些类似方法有什么不同）,会不会改变值本身?

数组常用方法

  • 声明方式
  1. 构造函数的声明方式： var arr = new Array();
  2. 字面量的声明方式：var arr=[];
  • shift() 从前面删除一个元素 会改变原有数组 返回删除的元素
  • pop() 从数组尾部删除一个元素 会改变原有数组 返回删除的元素
  • unshift() 从前面增加一个或多个元素 会改变原有数组 返回增加后数组的长度
  • push() 从后增加一个或多个元素 会改变原有数组 返回增加后数组的长度
  • reverse() 数组倒序 会改变原有数组 返回倒序后的数组
  • slice() 传入一个值代表从开始截取值的索引到数组结束位置 传入两个值第一个为开始截取值的索引 第二个为截取结束值的索引 可传入负值表 示从后面开始截取数组 不会改变原有数组 返回截取后的数组
  • sort() 数组排序 不传值默认全部转为字符串进行从小到大排序 传两个值，x,y x-y代表从小到大排序y-x代表从大到小排序 会改变原有数组 返回排序后的数组
  • indexOf() 查找数组是否存在某个元素 可传两个值第一个为要查找的元素, 第二个为开始查找的位置,不写默认从第一个开始查找 不改变原有数组 找到元素返回元素的索引值，没到到返回-1
  • forEach()数组循环 可传三个值 第一个为当前元素的值，第二个为当前元素的索引值，第三个为当前元素所属于的数组对象，不会改变原有数组 返回新数组;
  • map() 可传三个值 第一个为当前元素的值, 第二个为当前元素的索引值，第三个为当前元素所属于的数组对象，不会改变原有数组 返回一个新数 组，数组中的元素为原始数组元素调用函数处理后的值
  • filter() 数组过滤 可传三个值 第一个为当前元素的值，第二个为当前元素的索引值，第三个为当前元素所属于的数组对象，不会改变原有数组 返 回过滤后的新数组;
  • splice() 任意位置的增加、删除、修改 可传三个值 第一个为开始位置 第二个为改变的元素个数 第三个为替换的内容不写第三个代表删除 会改变原有数组 返回删除或替换的元素
  • toString()转换成字符串
  • fill() 填充数组 当传一个值的时候：改值为填充的值 传三个参数时第一个为填充的值，第二个为开始位置，第三个为结束位置
  • join() 把数组拼接为字符串 传入一个值，传入值为拼接符，不写默认用逗号作为拼接符 不会改变原有数组 返回拼接后的字符串
  • join(0) 用0拼接
  • concat() 合并数组 可传多个参数 不会改变原有数组 返回合并后的新数组
  • length 返回数组长度

字符串

  • 声明方式
  • indexOf()查找是否存在某个元素 可传两个值第一个为要查找的元素 第二个为开始查找的位置 不写默认从第一个开始查找 找到元素返回元素的索引值，没找到返回-
  • charAt() 传一个值（索引） 返回对应位置的字符串
  • replace()替换 传入两个值 前面的值替换成后面的值 返回替换后的字符串 不会改变原有字符串
  • slice()截取 返回截取的字符串 不会改变原有字符串 传入两值代表从开始位置到结束位置 支持传入负数 代表从后面开始截取
  • split()字符串分割为数组 ‘’代表把字符串分割为数组 传入其他值时，以该值为分割线将字符串分割成数组 不会改变原有字符串
  • substr()截取 返回截取的字符串 不会改变原有字符串 传入两值第一个代表从开始位置第二个代表截取长度 支持传入负数
  • substring()截取 返回截取的字符串 不会改变原有字符串 传入两值代表从开始位置到结束位置 默认把最大值当成结束位置最小值当成开始位置 不支持传入负数
  • charCodeAt()传入一个值 返回对应位置字符的ASCII编码
  • concat() 可传入多个值传入的值为要拼接的内容 字符串拼接 不会改原有字符串 返回拼接后的字符串1

  • lastIndexOf()返回一个指定的字符串值最后出现的位置，如果指定第二个参数 start，则在一个字符串中的指定位置从后向前搜索。找到元素返回元素的索引值，没找到返回-1

  • length 返回字符串的长度

  • repeat() 字符串重复 不会改变原有字符串 传入一个值代表重复次数
  • toLowerCase()英文字母全部转换成小写 不会改变原有字符串
  • toUpperCase()英文字母全部转换成大写 不会改变原有字符串
  • trim()去除字符串前后空格

日期对象

  • 声明方式

  • var date=Date() 返回当前系统时间，且为字符串类型 不支持传参

  • var date= new Date() 不写返回当前系统时间为对象类型 传入一个数字代表格林泥治时间加上传入的毫秒数
  • 传入一个字符串返回年 传入多个数字或者多字符时返回年月日时分秒 超出月份或者日期自动加1
  • 传入日期格式时 超出不会进一显示Invalid Date
  • getDate()返回当前日期
  • getDay()返回星期（0-6）0为星期天
  • getFullYear() 返回当前年份
  • getHours()返回当前小时
  • getMilliseconds()返回当前毫秒
  • getMinutes()返回当前分钟
  • getMonth()返回当前月份0-11（0代表1月）
  • getSeconds()返回当前秒
  • getTime()返回当前日期到格林尼治时间的毫秒数
  • setDate()设置日期
  • setFullYear()设置年
  • setHours()设置小时
  • setMilliseconds()设置毫秒
  • setMinutes()设置分钟
  • setMonth()设置月份
  • setSeconds()设置秒数
  • setTime()设置当前日期到格林尼治时间的毫秒数
  • toLocaleString() 返回年月日时分秒
  • toLocaleDateString()返回年月日
  • toLocaleTimeString()返回时分秒，分上午，下午

Math

  • abs()取绝对值
  • cbrt()开立方根
  • ceil()向上取整
  • floor()向下取整
  • max()传多个参数，选取传入值中的最大值
  • min()传多个参数，选取传入值中的最小值
  • pow()传两个参数x,y 代表x的y次幂
  • random()无参数，[0,1)之间的随机小数
  • round()四舍五入
  • sqrt()开平方根

关于BOM

  • BOM是什么？
  • 浏览器对象模型
  • location 浏览器的导航栏
  • host: 域名+端口号
  • hostName: 域名
  • port: 端口号
  • href: 完整路径
  • search: ？开始到结尾或到#结束（不包括#）
  • hash: #及后面的内容
  • pathName: 路径地址
  • protocol: 传输协议 http:（默认端口号80） https:(默认端口号443)
  • reload() 刷新页面
  • assign() 跳转页面，有历史记录

  • replace() 跳转页面，无历史记录

  • history 历史记录

  • back() 不传参 返回
  • forward() 不传参 前进
  • go() 不传代表刷新一次，传参1代表前进一次2代表前进两次 -1代表后退一次-2代表后退两次
  • length 获取历史记录条数
  • navigator 浏览器基本信息
  • userAgent 判断浏览器的版本信息

获取元素节点的方式？

  • 通过Id获取：document.getElementById() 只能返回一个dom对象
  • 通过class获取：document.getElementsByClassName() 统一返回类数组
  • 通过标签名称获取：document.getElemensByTagName(‘标签名’) 无论有无都会返回一个类数组
  • 通过name获取：document.getElementsByName(‘name的名字’) 返回类数组
  • 通过css选择器获取：document.querySelectorAll(‘’) 返回所有 返回类数组
  • document.querySelector(‘’) 找到第一个即返回 返回类数组
  • 获取下一个元素同级节点：nextElementSibling
  • 获取下一个节点同级：nextSibling
  • 获取上一个元素同级节点：previousElementSibling 没有返回null
  • 获取父节点：ParentNode
  • ParentElement 两者无区别
  • 获取子元素节点：Children
  • 获取所有的子节点:ChildNodes

  • 只有元素节点才有父节点和子节点，文本节点和注释节点无父子节点

  • 那几种获取的是集合？

  • 通过class获取：document.getElementsByClassName() 统一返回类数组
  • 通过标签名称获取：document.getElemensByTagName(‘标签名’) 无论有无都会返回一个类数组
  • 通过name获取：document.getElementsByName(‘name里的名字’) 返回类数组
  • 通过css选择器获取：document.querySelectorAll(‘’) 返回所有 返回类数组

  • document.querySelector(‘’) 找到第一个即返回 返回类数组

  • 获取的集合可以使用数组的方法么？

  • 不可以用数组方法
