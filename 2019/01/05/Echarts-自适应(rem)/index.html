rem 优缺点

  1. 布局的自适应代码量少，适配简单
  2. 因为是根据 ui 稿等比缩放，当大屏跟 ui 稿的比例不一样时，要单个做字体、间距、位移的适配

Echarts自适应

Echarts图表随屏幕的大小改变
