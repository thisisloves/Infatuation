parseInt()取整

  • parseInt();该方法从左向右，检测数字或字符串，当第一次发现一个不为数字的字符时，把之前的所有进行返回，如果字符串中第一个就不是数字，那么返回NaN
  • var str=”12323”; var num=parseInt(str); console.log(parseInt(str));console.log(num); //字符转数字
  • parseInt(“123abc”);a做为第一个不是数字的字符；返回a之前的所有值，则该方法返回123
  • parseInt(“abc123”);a做为第一个不是数字的字符，且处在字符串的第一位，则判定该字符串无法转换为数字返回NaN

parseFloat()取小数

  • parseFloat();该方法检索数字或字符串后面第一个.后的不为数字的字符，并对之前所有的结果进行返回，如果第一个字符不为数字，那么返回NaN
  • parseFloat(“123.456”);返回123.456
  • parseFloat(“123.4a56”);返回123.4
  • parseFloat(“123.00”);返回123
  • parseFloat(“123.0a56”);返回123

Number() 所有字符数字转换成数值数字

  • var a=123;
  • var b=”123”;
  • var c=”123b”;

  • var d=”12.21100”

  • console.log(Number(a)) //123

  • console.log(Number(b)) //123
  • console.log(Number(c)) //NaN
  • console.log(Number(d)) //12.211

toString() 转字符

  • num.toString()数字转字符

  • var c=”123b”;
  • var d=”12.21100”
  • console.log(a.toString()); //123
  • console.log(b.toString()); //123

  • console.log(d.toString()); //12.21100

toFixed() 转换成字符串的同时保留n位小数

  • num.toFixed(n) 转换成字符串的同时保留n位小数
  • e=123.222
  • f=123.300
  • g=”123.8f0”
  • console.log(e.toFixed(1)); //123.2
  • console.log(f.toFixed(1)); //123.3
  • console.log(g.toFixed(1)); // 报错

Nan

  • Nan：不是一个数字的数据值型数据，代表非法运算或者转换的结果 not a number 是一种特殊的Number类型，代表意外转换的数字，NaN和任何东西都不相等。
  • 厉害到自己都不等于自己
  • var str=”a2344”
  • console.log(typeof parseInt(str)) //结果Nan
  • var str=”2344a”
  • console.log(typeof parseInt(str)) //结果2344
  • console.log(Nan==Nan) //false

unll 和undefined

  • defined:定义

  • null 表示一个值被定义了，定义为“空值”；

  • undefined 表示根本不存在定义。

  • 所以设置一个值为 null 是合理的，如

  • objA.valueA = null;
  • 但设置一个值为 undefined 是不合理的

if-else 隐式类型转换

  • if(1){console.log(“真”);}else{console(“假”);} //输出真
  • if(布尔，计算结果也是布尔型){}
  1. 非0为true,0为flase 发生了隐式类型转换成了布尔型boolean
  2. 非空字符为true ,空字符为false
  3. undefined和Nav和null为false
  4. 对象为真{}
  5. 函数为真function()

+ 隐式类型转换

  • console.log(1+true); //2
  • console.log(1+false); //1
  • console.log(1+undefined); //NaN
  • console.log(1+NaN); //NaN
  • console.log(1+null); //NaN
  • console.log(1+{name:”admin”}); //1[object object]
  • console.log(1+[3,4,5]); //1数组的值：13，4，5

switch（交换） case(事件)

  1. case 的穿透性：当上一个case判断成功后，后面的case不在判断直接执行 自己 的语句。所以需要加上break。
  2. switch:不会发生隐式类型转换。
  3. 判断具体数据使用switch。
  4. 使用 default 关键词来规定匹配不存在时做的事情：不在0-6范围内，就输出default。

逻辑运算符，实现简单判断

  • 只要出现true就是true,1为1
  • console.log(true|| false); //t
  • console.log(true|| 0); //t
  • console.log(1|| 0); // 1

  • console.log(true|| 1}; //t

  • console.log(true||true}; //t

  • console.log(true|| 3); //t
  • console.log(1|| 3); //1

三目运算符

  • 基本语法：expression?sentence1 :sentence2
  • 判断expression是否为真，若为正执行sentence1否则执行sentence2
  • var a = 100;
  • var b = 200;
  • a>b ? document.write(“a比b大”):document.write(“a比b小”); //输出a比b小
