编程语言

  • javascript:编程语言，脚本语言，解释性的语言
  • js万物皆对象 document
  • BOM:浏览器对象模型
  • DOM:文档对象模型
  • browser object model 浏览器对象模型
  • ECMAScript 简称ES,语法规范，法律，规定的变量数据函数等等

js组成

  • BOM:浏览器对象模型
  • DOM:文档对象模型
  • ECMAScript 简称ES,语法规范，法律，规定的变量数据函数等等

js引入

  • js文件内，script标签的src引入 推荐外部：便于维护

注释

  • 单行注释 //
  • 多行注释 / /

数据类型

  • 数值型(number)：1232
  • 字符型(string)：只要放在”” 和’’ 中的就是字符
  • 布尔型(boolean)：true/false
  • 未定义型(undefined:undefined
  • 对象型(object)：{}十有八九是对象
  • 数组型(array)：[]十有八九是数组

变量

  • 创建变量： var a=10; var b=”hello”; var c=”true”;
  • var 是声明
  • 检测变量类型 var a=”20” alert(typeof a);

变量命名规则

  • 1.前缀就能表示出这个变量的类型
  • 2.开头为字母,下划线_ 和$
  • 3.关键字和保留字不能保留
  • 赋值： =

打印信息

  1. alert(1); //弹出框打印 //bom //暂替程序执行，不方便
  2. document.write(1) // 将信息打印到页面 //dom //页面布局会影响
  3. document.write(“123“) 页面会解析输出123 $lt不解析
  4. console.log(100) // 将信息打印到浏览器控制台 //bom //推荐

算术运算符

  • 算术运算符：+ - * / %(取余)
  • 在算术运算符隐式类型转换：+号两边，只要出现一个字符，那么结果就是字符string， 而遇到 - * / % 会转成number数值
  • var a=10;var b=”100”; console.log(typeof(a+b)) 输出string; 字符拼接

  • var a=10;var b=”100”; console.log(typeof(a-b)) 输出number; //无隐式转换

    关系运算符

  • 关系运算符： > < >= <= == ===

  • var a=100; var b=20; console.log(a>b); 输出true;
  • var a=”100”; var b=”20”; console.log(a>b) 输出flase
  • // 字符比较：逐位比较 1>2 false 停止
  • var a=”100”; var b=20; console.log(a == b) true
  • var a=”100”; var b=20; console.log(a === b) false
  • // 3个等于不存在隐式转换不仅比较值还比较类型

逻辑运算符

  • 或or || ,且and &&, 非not !
  • console.log(flase||true); true
  • console.log(true||true); true

  • console.log(false||false); false

  • console.log(flase && true); false

  • console.log(true && true); true

  • console.log(false && false); false

  • console.log(! true); fasle

  • console.log(!false); true

  • //结合关系运算符

  • // 被4整除，不能被100整除
  • // 被400整除
  • year%4 == 0 && year%100 !=100 ||year%400==0
  • console.log(year%4 == 0 && year%100 !=100 ||year%400==0)

赋值运算符

  • 赋值运算符： = += *= /= %=
  • var a=10; //10
  • a=a+10; 简写 a+=10; //20
  • ++ 自增
  • – 自减

  • var a=100; a–; console.log(a); //a=9

  • 后自增

  • var a=10; console.log(a++); console.log(a); // 先使用，后运算

  • 前自增

  • var a=10; console.log(++a); console.log(a); //先运算，后使用

  3. 循环(while)
