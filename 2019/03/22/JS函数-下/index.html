return

  • 让函数有返回值的关键字; 终止函数;
  • 其实每个函数都有返回值，只不过，没有return的函数，返回的是undefined
  • 有return的，返回的 return的值
  • 使用场景
  • 当函数需要有返回值的时候，再加return;
  • 处理数据的函数，处理好的数据，需要再次使用再次调用，需要return
  • 实现功能的函数，不需要返回值

变量作用域

  • 变量作用域：变量作用的区域，生效的区域

作用域分类

  • 全局变量：不在任何函数内，就是全局作用域
  • 局部变量：任何一个函数都是一个局部作用域
  • 读取规则：局部可以使用全局，全局不能使用局部
  • 变量生命周期：局部在函数内部，使用完就删除
  • 全局变量：永生,一直存在不删除

声名提升

  • 在函数中，没有定义变量之前打印a,得到undefined,而并不是not defined
  • 说明此时变量已经被声明，但是没有被赋值。

  • 这种现象被称为变量的声名提升，所有使用var声明的变量都会在当前作用的作用域最开始的部分被声明。

  • 函数的声明提升：整体提升

构造函数

  • var str=new String(“123”); //构造函数方式生成
  • console.log(str); //[1,2,3,…]
  • console.log(typeof str) //object
  • 构造函数只会产生对象，构造函数是用来构造对象的函数
