随机背景颜色

随机点名

定时器

this

  • this:这个：指针，指向当前函数的执行对象(谁执行了当前函数,this就是谁)
  • 当前函数的执行上下文空间
  • 严格模式：use strict

BOM

  • BOM：浏览器对象模型
  • BOM提供了独立于内容于浏览器窗口进行交互的对象
  • BOM缺乏标准，javascript语法的标准化组织是ECMA,DOM的标准化是W3C
  • BOM主要用于管理窗口于窗口之间的通讯，因此核心对象是window(window一般简写)

window对象（内置对象）

  • window对象是浏览器内置的一个对象，相当于BOM的一个抽象，是js中全局对象的寄集地

  • window对象的常见方法：

  • alert(); //弹出信息框
  • prompt(对话框消息); //弹出输入框，点击确认，返回字符串，点击取消，返回null
  • confirm(提示信息); //点击确定返回true，点击取消返回false
  • close(); //关闭浏览器(火狐不支持)，但都支持通过别的网页打开的新页面关闭，不允许关闭非脚本打开的页面
  • open(url,name,feature,replace);
  • // url,可选，要在新窗口显示的文档的URL
  • // name,可选，声明了新窗口的名称
  • // feature，replace自行扩展
  • // 谷歌会默认把系统自动打开的新网页阻止掉，但不阻止通过事件打开的网页

window对象下的内置子对象

  1. history对象：该对象包含浏览器访问过的url
  • 该对象的属性：
  • history.length; //返回历史记录的数量
  • 该对象的方法：
  • history.back(); //后退加一个url
  • histroy.forward(); //前进，需要后退一下之后，才会有前进的方向
  • histroy.go(num); //参数为正，前进相应的数目，为负，后退相应的数目，为0
  1. location对象：包含当前url的相关信息，而history对象不能具体反映url的相关信息
  • 该对象的属性：
  • location.href; //设置或者返回完整的url
  • location.search; //返回url?后面的查询部分
  • location.hash; 是一个可读的字符串，是url的锚部分(从#开始的部分)哈希
  • 该对象的方法：
  • location.reload(); //刷新页面的方法，一般情况下，传递一个true,不使用缓存刷新。
  1. navigator对象
  • navigator.appName; // 返回当前浏览器的名称
  • navigator.appVersion; //返回当前浏览器的版本号
  • navigator.platform //返回计算机操作系统

onload事件

  • 当文档加载完成后执行一些操作
  • window.onload=function(){console.log(“页面加载完成”)}

DOM的选择器

  1. getElementById(id) //获取指定元素的ID元素
  2. getElementsByTagName() //获取相同元素的节点列表，通过标签名选择元素，返回值是一个数组
  3. getElementsByName() //通过name值获取元素，返回值是数组，通常用来获取有name的input值
  4. getElementsByClassName() //通过class名获取元素，返回值是数组
  • //不是所有的标签都有name值 ,在低版本的浏览器中getElementsByname()和getElementsByClassName()有兼容性

window.onscroll/onresize事件

  • onscroll当页面发生滚动时执行一些操作

  • window.onscroll=function(){ console.log(1) //当页面发生滚动时，打印1}

  • onresize当窗口发生大小改变的时候执行一些操作

  • window.onresize=function(){ console.log(1); //当页面发生大小改变时候，打印1}
  • 页面滚动条距离顶部的距离
  • document.documentElement.scrollTop
  • 页面滚动条距离左边的的距离
  • document.documentElement.scrollLeft
