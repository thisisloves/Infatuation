linux静态ip设置

  • 先进入 /etc/sysconfig/network-scripts/ifcfg-网卡
  • 以下两个的值如果不清楚的话，可以打开你本地的电脑，–》控制面板\所有控制面板项\网络和共享中心\本地连接\详细信息
  • 保存退出
  • 重启网络服务
  • service network restart
  • 检查一下状态
  • ping baidu.com
  • 我遇到两个情况：

(1) ping通了，oh yeah！

(2) 报错：ping:unknow host baidu.com

  • 解决方法是：
  • dhclient
  • 敲了这个命令后，再ping一次

  • 还有人是这样：配置静态IP之后reboot

  • 还有一个情况就是，ping 外网IP可以，但是无法ping域名。我的解决办法是：设置DNS

  • vim /ect/resolv.conf

  • nameserver 114.114.114.114 //这个值我是在本地连接的状态信息里找到的

  • 保存之后退出，再ping！

liunx防火墙

  • centos7
  • 启动： systemctl start firewalld
  • 关闭： systemctl stop firewalld
  • 查看状态： systemctl status firewalld
  • 开机禁用 ： systemctl disable firewalld
  • 开机启用 ： systemctl enable firewall
  • centos 6
  • 关闭防火墙： network iptables stop
  • 重启防火墙：network iptables restart
  • 永久关闭防火墙：chkconfig iptables off
