redux

  • redux是javascript的状态管理器，提供可以预测化的状态管理器。
  • redux试图让state的变化变得可以预测
  • 核心概念
  1. model
  2. action
  3. reducer
  • 三大原则
  1. 单一的数据源：整个应用的state被存储在一棵object tree中，并且这个object tree 只存在于唯一一个store中
  2. state是只读的：唯一改变state的方法就是触发action,action是一个用于描述以发生事件的普通对象
  3. 使用存函数来执行修改：为了描述action如何改变state tree,你需要编写reducers

  • 引入createStore方法，创建纯函数reducer,生成store，暴露store
  • index.js引入
