准备

  • 安装 node.js 链接：https://nodejs.org/en/
  • 安装 git 链接：https://git-scm.com/downloads
  • 安装npm 使用淘宝镜像 npm install -g cnpm –registry=https://registry.npm.taobao.org
  • 检查版本
  • node -v
  • npm -v
  • hexo -v

配置

  • 切换到文档目录
  • npm install hexo-cli -g
  • hexo init
  • npm install
  • hexo clean
  • hexo server
  • hexo d -g

遇到的问题

  • 问题1 hexo s 不成功 解决方法：重新安装node.js和hexo
  • 问题2 hexo d 不成功 解决方法：github上的仓库将只读改为可读可写
  • 问题3 github显示空白无效果 解决方法：_config.yml 路径改成相对路径relative_link: true

  • 问题4 字体图标无效果 解决方法：使用CDN源字体

  • 更改head.ejs
